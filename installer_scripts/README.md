# Scripts auxiliares

Esta carpeta agrupa scripts que no son el punto de entrada principal.

- `install_linux.sh`: crea `.venv`, instala dependencias y descarga modelos.
- `download_models_linux.sh`: reintenta la descarga de modelos.
- `uninstall_linux.sh`: elimina dependencias locales y, si se confirma, modelos o logs.
- `requirements.txt`: dependencias Python de la aplicacion.

Para uso normal en Linux, ejecutar `./run_app.sh` desde la raiz.
