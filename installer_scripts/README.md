# Scripts auxiliares

Esta carpeta agrupa scripts que no son el punto de entrada principal.

- `install_windows.bat`: crea `.venv`, instala dependencias y descarga modelos.
- `download_models_windows.bat`: reintenta la descarga de modelos.
- `uninstall_windows.bat`: elimina dependencias locales y, si se confirma, modelos o logs.
- `requirements.txt`: dependencias Python de la aplicacion.

Para uso normal en Windows, ejecutar `run_app.bat` desde la raiz.
